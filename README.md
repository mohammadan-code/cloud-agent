# Cloud Agent

A workspace configured with opinionated Cursor AI rules that enforce honest, creative, and unbiased agent behavior.

## Cursor Rules

This repo ships with `.cursor/rules/` that shape how the AI agent behaves:

| Rule | Purpose |
|------|---------|
| `honesty-first.mdc` | No hallucination. Admit uncertainty. Never fabricate. |
| `creative-unbiased.mdc` | Think from multiple angles. Challenge assumptions. No default bias. |
| `straight-talk.mdc` | Direct answers. No filler. No sugar-coating. |

## Setup

1. Clone this repo
2. Open in Cursor
3. The rules activate automatically — no config needed
